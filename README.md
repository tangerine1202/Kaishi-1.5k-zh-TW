# Kaishi 1500 to PDF

Convert Kaishi 1500 to PDF using Typst.

## Usage

```bash
uv run convert_kaishi.py
# if the font is not found, you can add the font path to the command
typst compile --font-path=/Users/<user>/Library/Fonts kaishi-1500.typ
```
